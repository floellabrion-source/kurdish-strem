# Kurdish Stream 🎬

پلاتفۆرمی ستریمینگی کوردی - مانند Netflix

## ڕاکردن

### پێویستییەکان
- Node.js (v18+)
- npm

### هەنگاوی یەکەم: دامەزراندنی پاکێجەکان

**ئتریمان (Server):**
```powershell
cd "C:\Users\BSMA CO\Downloads\kurdish-stream\server"
npm install
```

**فرۆنتێند (Client):**
```powershell
cd "C:\Users\BSMA CO\Downloads\kurdish-stream\client"
npm install
```

### هەنگاوی دووەم: ڕاکردنی سێرڤەر و فرۆنتێند

**تێرمینالی یەکەم (Server):**
```powershell
cd "C:\Users\BSMA CO\Downloads\kurdish-stream\server"
npm start
```

**تێرمینالی دووەم (Client):**
```powershell
cd "C:\Users\BSMA CO\Downloads\kurdish-stream\client"
npm run dev
```

بەرنامەکە دەکرێتەوە لەسەر: **http://localhost:5173**

## بەکارهێنان

1. بڕۆ بۆ `/admin` بۆ زیادکردنی فیلمی نوێ
2. فیلمەکان بڕۆ بۆ **uploads/movies/** خەزن دەبن لەناو کۆمپیوتەرت
3. بۆ بینین بڕۆ بۆ لاپەڕەی سەرەکی
